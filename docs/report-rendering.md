# 报告校验与 HTML 渲染

第 17 阶段不再调用 AI。`ReportService` 将已持久化的确定性决策、AI/规则综合结果、
证据和原始来源组装成 `Report`，先检查上下文、规则版本、Prompt 版本、Evidence ID、
允许动作和非强制金额/比例边界，然后才渲染。当前模板版本为 `report-html-1.0.0`。

报告包含数据时效标记、建议与边界、多空证据、因果链、未确认信息、失效条件、
原始链接、来源健康状态、规则/Prompt/模型/模板版本和明确的非交易声明。超过 30 小时的
数据在顶部标为过期。

标题、证据、摘要、因果链和来源名称全部通过 HTML 转义，不解析 Markdown 或原始 HTML。
外链只允许无凭据的 HTTP(S) 80/443 端口，并固定使用
`rel="noopener noreferrer nofollow"`。文档内嵌严格 CSP，禁用脚本、图片、表单、框架和外部资源。

渲染后的 UTF-8 HTML、SHA-256、输入快照和模板版本写入 `reports`。报告行禁止原地更新
或删除；历史对比直接读取当时快照，不用新模板重新渲染旧报告。
