# ADR 0001：系统只提供建议，不执行交易

## 状态

已接受，2026-07-17。

## 背景

用户希望工具帮助收集证据、理解风险并生成投资建议，但所有实际申购、定投调整、赎回和其他仓位操作都由用户独立判断和完成。

## 决定

- 系统不保存支付宝或券商凭据；
- 系统不实现交易适配器；
- 系统不实现申购、赎回、下单、调仓或账户控制 API；
- `SMALL_ADD`、`PAUSE_ADDING`、`REDUCE` 等词只能是报告中的建议标签；
- 系统可以解释建议依据和提供非强制的参考边界，但不能执行建议；
- 用户始终拥有最终判断权和操作权。

## 后果

- 安全边界更清晰，外部新闻和 AI 无法影响真实账户；
- 测试必须持续验证不存在交易路由；
- 未来即使扩展数据源、AI 或报告，也不得隐式加入交易执行能力；
- 如未来需求发生变化，必须新建架构决策记录并重新进行安全与合规评审，不能直接修改本决定。
